Portfolio

Recently, I conducted research with Prof. Richard Furuta and Prof. Filipe Castro on The Nautical Archaeology Digital Library. We researched the best way to add images/paintings of ships to a database and how to crop and highlight certain parts of the ships to assign more information to it. Furthermore, we researched the best and the easiest way to add filtering capabilities to those images.


Pre-Adventure

For the final project in my Programming Studio class, our group made a stress free travel planning website. When planning a trip you often need to search through several applications and websites to gather all the information and guidance you need. Such a process is tedious and time-consuming often feeling more like research/work which is why our application will combine all the information you might need in one place so the user is left with a stress-free planning experience. With all the information in one place, the user will no longer have to scour the internet for random sites that have bits of information, instead the user will be free to focus on what is important, how to make their trip/vacation experience the best it can be.


Virus Simulation (Python)

For the final project in my Engineering Lab Computation, my group made a simulation of how a virus would spread depending on what parameters are passed in. This project takes a population size and a simulation duration from the user. Then, it runs the simulation that will track an infection spread throughout a population. At each time step and at the end, statistics are reported. This project was implemented in Python.


Data Structure Implementations (C++)

During my Data Structures & Algorithm course, I implemented a singly linked list structure, a doubly linked list structure, different implementations of Stack, different implementations of the Priority Queue, etc. Learning about the different advantages and disadvantages of different structures was very helpful. Learning about when to use what structure is knowledge that I will use for the rest of my life.


Simplified Instructional Set Architecture (Python, y86 & Logisim)

For the final project of my Computer Organization class, my teammate and I built a simplified ISA based upon the y86 syntax. This project takes in code in Y86 syntax, transforms it into hex code using Python, and executes it in a CPU built in Logisim. This project helped me understand how a CPU actually works, which is essential knowledge for a programmer. The image is shown on the left. Hover over to enlarge. Click here for the photo.
